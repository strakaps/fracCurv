Fractal Curvature

Peter Straka

02/01/2015

This page illustrates calculations from the paper Estimation of fractal dimension and fractal curvatures from digital images. After loading the workspace fracCurv.RData, the data sets for the six fractal sets (gasket, carpet, modcarpet, quadrate, triangle, supergasket) become available. These were calculated using the Java library GeoStoch; see here for the main method. The images have resolution \(3000 \times 3000\), or \(1500 \times 1500\) if the number 2 appears in the data set name.

Non-arithmetic fits

The method LRE (linear regression estimate) is called as follows:

fracCurv(gasket,arithmetic = FALSE)

This results in a linear fit to the data \(y_{kj} := \log \left( \varepsilon_j^{-k} |C_k(F_{\varepsilon_j})|\right), \quad k=0,\ldots, d\). The output contains estimated fractal dimension, curvatures and fitted linear model.

fracCurv(gasket,arithmetic = TRUE)

Fits with known dimension

Assuming the dimension is known, a fit is performed e.g. as

fracCurv(gasket[,c(1,2)],arithmetic = FALSE, frac.dim = log(3)/log(2))

Note that only one characteristic can be fitted at a time, here for \(k = 0\). If dimension and period are known (for arithmetic fractals) one calls e.g.

fracCurv(gasket[,c(1,2)],arithmetic = TRUE, frac.dim = log(3)/log(2), period = log(2))
